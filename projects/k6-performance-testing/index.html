Kata 1: Basic Load Test Link to heading

  • Objective: Verify server responds correctly to a GET request
  • Endpoint: https://jsonplaceholder.typicode.com/users
  • Validates: HTTP status 200

Kata 2: Response Time Validation Link to heading

  • Objective: Measure response time and verify it’s under threshold
  • Endpoint: https://jsonplaceholder.typicode.com/users
  • Validates: Response time < 200ms

Kata 3: Parallel Users Link to heading

  • Objective: Simulate multiple users making parallel requests
  • Endpoint: https://jsonplaceholder.typicode.com/users
  • Config: 10 virtual users, 30 seconds

Kata 4: Stress Test Link to heading

  • Objective: Long-duration stress test for stability
  • Endpoint: https://jsonplaceholder.typicode.com/users
  • Config: 10 VUs, 10 minutes

Kata 5: Authentication Link to heading

  • Objective: API request with Bearer token
  • Endpoint: https://httpbin.org/bearer
  • Validates: Authentication token

Kata 6: Error Handling Link to heading

  • Objective: Handle 404 error responses
  • Endpoint: https://jsonplaceholder.typicode.com/invalid-endpoint
  • Validates: 404 status code

Kata 7: Dynamic Data Validation Link to heading

  • Objective: Validate response data matches expected values
  • Endpoint: https://jsonplaceholder.typicode.com/users/1
  • Validates: User name is “Leanne Graham”

Kata 8: Custom Metrics Link to heading

  • Objective: Custom metrics for performance tracking
  • Uses: Trend and Rate metrics

Kata 9: Complete Load Test Link to heading

  • Objective: Full load test with custom reports
  • Config: 10 VUs, 5 minutes, summary reporting
  • Output: JSON summary report
