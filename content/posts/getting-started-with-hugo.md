# Getting Started with Hugo

Hugo is a fast and flexible static site generator. In this post, I'll show you how to create a personal website using Hugo.

## Why Hugo?

- **Speed**: Hugo is incredibly fast
- **Flexibility**: Wide range of themes
- **No dependencies**: Single binary installation
- **Markdown**: Write content in Markdown

## Installation

```bash
# On macOS
brew install hugo

# On Linux
sudo apt-get install hugo

# On Windows
choco install hugo -confirm
```

## Creating a New Site

```bash
hugo new site my-website
cd my-website
hugo server
```

Hugo is an excellent choice for building static websites. Stay tuned for more tutorials!
