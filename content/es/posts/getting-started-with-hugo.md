# Comenzando con Hugo

Hugo es un generador de sitios estáticos rápido y flexible. En esta publicación, te mostraré cómo crear un sitio web personal usando Hugo.

## ¿Por qué Hugo?

- **Velocidad**: Hugo es increíblemente rápido
- **Flexibilidad**: Amplia gama de temas
- **Sin dependencias**: Instalación de binario único
- **Markdown**: Escribe contenido en Markdown

## Instalación

```bash
# En macOS
brew install hugo

# En Linux
sudo apt-get install hugo

# En Windows
choco install hugo -confirm
```

## Crear un Nuevo Sitio

```bash
hugo new site mi-sitio
cd mi-sitio
hugo server
```

## Conclusión

Hugo es una excelente opción para construir sitios web estáticos. ¡Mantente atento para más tutoriales!
