# Sitio de Portafolio Personal

Un sitio de portafolio personal moderno y responsivo construido usando el generador de sitios estáticos Hugo.

## Características

- Diseño completamente responsivo
- Soporte multilenguaje (Inglés, Español, Portugués)
- Alternar modo oscuro/claro
- Optimizado para SEO
- Carga rápida

## Tecnologías Usadas

- Hugo
- HTML5/CSS3
- Font Awesome Icons
- GitHub Pages (alojamiento)
