# Site de Portfólio Pessoal

Um site de portfólio pessoal moderno e responsivo construído usando o gerador de sites estáticos Hugo.

## Características

- Design totalmente responsivo
- Suporte multilíngue (Inglês, Espanhol, Português)
- Alternância modo escuro/claro
- Otimizado para SEO
- Carregamento rápido

## Tecnologias Usadas

- Hugo
- HTML5/CSS3
- Font Awesome Icons
- GitHub Pages (hospedagem)
