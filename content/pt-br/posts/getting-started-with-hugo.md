# Começando com Hugo

Hugo é um gerador de sites estáticos rápido e flexível. Nesta postagem, mostrarei como criar um site pessoal usando Hugo.

## Por que Hugo?

- **Velocidade**: Hugo é incrivelmente rápido
- **Flexibilidade**: Ampla gama de temas
- **Sem dependências**: Instalação de binário único
- **Markdown**: Escreva conteúdo em Markdown

## Instalação

```bash
# No macOS
brew install hugo

# No Linux
sudo apt-get install hugo

# No Windows
choco install hugo -confirm
```

## Criando um Novo Site

```bash
hugo new site meu-site
cd meu-site
hugo server
```

## Conclusão

Hugo é uma excelente escolha para construir sites estáticos. Fique atento para mais tutoriais!
